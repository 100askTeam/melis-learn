# Pin assignments

The table below shows the E-paper display pin assignments.

 E-paper display Pin     | SPI pin | DShanPi-D1s pin      |  Notes
-------------------------|---------|----------------------|-------------
 BUSY                    |         | GPIO_PG1             |
 RESET                   |         | GPIO_PG2             |
 D/C                     |         | GPIO_PG4             |
 CS                      | CS      | GPIO_PD10            |
 SCLK                    | SCK     | GPIO_PD11            |
 SDI                     | MOSI    | GPIO_PG12            |
